イベントの投稿申請

コントロールパネルのこの部分で、ユーザーが投稿したイベントの編集・削除・承認を行うことができます。削除・承認は該当するボタンをクリックしてから、「実行」をクリックしてください。

フィールド 説明
編集 ペンのアイコンをクリックすると、イベントを編集できます。Note that saving an edited event submission will approve it.
タイトル イベントのタイトルです。
開始 イベントの開始日です。
URL イベントのURLです。
削除 / 承認 イベントの削除/承認を選択します。
