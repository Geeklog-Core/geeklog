リンクの投稿申請

コントロールパネルのこの部分で、ユーザーが投稿したリンクの編集・削除・承認を行うことができます。削除・承認は該当するボタンをクリックしてから、「実行」をクリックしてください。

フィールド 説明
編集 ペンのアイコンをクリックすると、リンクを編集できます。 Note that saving an edited link submission will approve it
タイトル リンクのタイトルです。
カテゴリ リンクが投稿されたカテゴリです。
URL リンクのURLです。
削除 / 承認 リンクの削除/承認を選択します。
