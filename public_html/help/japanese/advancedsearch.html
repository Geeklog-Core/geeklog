検索オプション

高度な検索を使用することにより、すべての記事とコメントを検索して探しているものを見つけることができます。検索を実行するときに、少なくとも1つのフィールドを選択する必要があります。検索用のフィールドは以下の通りです。

フィールド 必須 説明
キーワード いいえ 検索したいキーワードで、例としては"Free"や"Fun"を(引用符は付けずに)入力します。

ドロップダウンメニューから検索を完全一致(入力した文字列そのまま)、単語をすべて含む、単語のいずれかを含むから選択できます。

ここにはワイルドカードも使用できます。"%"はゼロまたは1つ以上の文字に、"_"は1つの文字に一致します。
Titles Only No Check this box if you only want to search item titles
日付 いいえ 検索対象の日付を入力します。開始日と終了日を入力してください。
話題 いいえ 検索対象の話題をドロップダウンリストから選択します。
タイプ いいえ 検索の種類を選択します。選択できるのは「すべて」(サイト内の検索可能なものすべて)、または「記事」や「コメント」やだけでなくサイトで利用できる他のコンテンツ(静的ページなど)です。
所有者 いいえ 検索対象とする所有者を選択します。
Results No Select the number of search results you want to see per page
