記事の投稿申請

コントロールパネルのこの部分で、ユーザーが投稿した記事の編集・削除・承認を行うことができます。削除・承認は該当するボタンをクリックしてから、「実行」をクリックしてください。

フィールド 説明
編集 ここをクリックすると、イベントを編集できます。 Note that saving an edited story submission will approve it unless you save it with the draft flag set.
タイトル 記事のタイトルです。
日付 記事が投稿された日付です。
話題 記事が投稿された話題です。
削除 /承認 記事の削除/承認を選択します。
