Comment Submissions

This is the command and control quick moderation area for comments. In this area you can edit, delete or approve comments. Select any combination of deletes or approves and click the submit button.

Field Description
Edit Click this to edit the comment. Note that saving an edited comment submission will approve it
Title Title of the comment
Parent or Comment Depending on the type of comment, this will either contain the comment's text or a link to the parent object for which the comment was submitted
Date Date the comment was submitted
Delete / Approve Radio buttons to select for delete or approval of the story submission
User Name Name of the user who submitted the comment. For registered users, this will contain a link to their profile page
Auto-publish Comments? Check this box to allow any further comments by this user to be published without moderation. Only available for registered users
