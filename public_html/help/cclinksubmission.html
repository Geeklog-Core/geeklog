Link Submissions

This is the command and control quick moderation area for links. In this area you can edit, delete or approve links. Select any combination of deletes or approves and click the submit button.

Field Description
Edit Click this to edit the link. Note that saving an edited link submission will approve it
Title Title of the link
Category Category the link was submitted to
URL URL for the link
Delete / Approve Radio buttons to select for delete or approval of the link
