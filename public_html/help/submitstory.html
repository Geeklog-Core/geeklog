Submit a Story

When submitting information for use on this site we ask that you follow the following suggestions ...

  • Fill in all the fields, they're required
  • Provide complete and accurate information
  • Double check those URLs
Field Required Description
Username No If you want credit for the story, make sure you're logged in
Title Yes Title for the story you are submitting
Topic Yes Topic this story falls under
Story Yes This is the story. It can be separated into an introduction and a body part. You can use HTML
Post Mode Yes If this story contains HTML, make sure you select that as the post mode. If your story contains code examples and HTML examples, select plain text to ensure that the examples won't be converted
