Calendar Submissions

This is the command and control quick moderation area for events. In this area you can edit, delete or approve events. Select any combination of deletes or approves and click the submit button.

Field Description
Edit Click this to edit the event. Note that saving an edited event submission will approve it
Title Title of the event
Start Start date of the event
URL URL for the event
Delete / Approve Radio buttons to select for delete or approval of the event
