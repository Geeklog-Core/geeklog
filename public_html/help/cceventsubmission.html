Event Submissions

This is the command and control quick moderation area for events.  In this area you can edit, delete or approve events.  Select any combinitation of deletes or approves and click the submit button.
 
Field  Description
edit Click this to edit the event
Title  Title of the event
Start Start date of the event
URL URL for the event
Delete/Approve Radio buttons to select for delete or apprval of the event
