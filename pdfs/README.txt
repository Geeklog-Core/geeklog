This is the Geeklog PDF directory.  This directory is used to store system generated PDF files using 
Geeklog's built-in PDF generator.  You may or may not choose to use this directory to store your
PDF's...to see a complete list of configuration options for the Geeklog PDF feature please read the
PDF section in the Geeklog configuration file (config.php).  Regardless of where you choose to store
your PDF's, they should be (but aren't required) outside of your webtree and the directory must be
writeable by the same user your web server is running as.
